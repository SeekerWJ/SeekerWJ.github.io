所谓class就是

包含了多个函数的集体，并且其中也有定义好的变量(属性)

Class的优点是可以保存状态

Class就有点像是把多个函数进行封装

class中的函数称之为方法，变量称之为属性，对象称之为实例

类

1.创建和使用类

class Dog():

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sit(self):
        print(self.name.title() + " is now sitting")

    def roll_over(self):
        print(self.name.title() + "rolled over!")

my_dog = Dog('wuhu~~','2')
print(my_dog.age)
my_dog.sit()
#输出结果
2
Wuhu~~ is now sitting

2. 使用类和实例，默认和修改属性的值

class Car():
    def __init__(self,make,model,year):

- 修改属性值的途径

  1. 直接修改属性的值

    class Car():
    def __init__(self,make,model,year):
        self.make = make
        self.model = model
        self.year = year
        self.odometer_reading = 0

    def get_descriptive_name(self):
        long_name = str(self.year) + ' ' + self.make + ' ' + self.model
        return long_name

    def read_odometer(self):
        print("This car has " + str(self.odometer_reading) + " miles on it")

my_new_car = Car('audi','A6','2020')
print(my_new_car.get_descriptive_name())
my_new_car.read_odometer()

my_new_car.odometer_reading = 20#在这里直接修改属性的值了
my_new_car.read_odometer()
#输出结果
2020 audi A6
This car has 0 miles on it
This car has 20 miles on it

  2. 通过方法修改属性的值

    通过这种方法，无需直接访问属性，可将值直接传递给一个方法，由该方法来对需要改变的属性赋值，这种操作更为直观方便。

    这种方法在对修改属性值的时候也可以对其做一些其他操作，比如当修改的值小于原来的值时，禁止修改等…

    class Car():
    def __init__(self,make,model,year):
        self.make = make
        self.model = model
        self.year = year
        self.odometer_reading = 0

    def get_descriptive_name(self):
        long_name = str(self.year) + ' ' + self.make + ' ' + self.model
        return long_name

    def read_odometer(self):
        print("This car has " + str(self.odometer_reading) + " miles on it")

    #定义一个改变属性的方法    
    def updata_odometer(self,mileage):
        self.odometer_reading = mileage

my_new_car = Car('audi','A6','2020')
print(my_new_car.get_descriptive_name())
my_new_car.read_odometer()

#直接调用该方法来进行赋值
my_new_car.updata_odometer(23)
my_new_car.read_odometer()
#输出结果
2020 audi A6
This car has 0 miles on it
This car has 23 miles on it

3. 继承

当需要编写的类是另一个现成类的特殊版本，可以使用继承。

子类继承了父类的属性和方法，同时也可以拥有属于自己的特殊的属性和方法

class Car():
    def __init__(self,make,model,year):
        self.make = make
        self.model = model
        self.year = year
        self.odometer_reading = 0

    def get_descriptive_name(self):
        long_name = str(self.year) + ' ' + self.make + ' ' + self.model
        return long_name

    def read_odometer(self):
        print("This car has " + str(self.odometer_reading) + " miles on it")

    def updata_odometer(self,mileage):
        self.odometer_reading = mileage

class ElectricCar(Car):
    def __init__(self,make,model,year):
        #将父类与子类关联起来，让子类包含父类的所有属性，方法
        super().__init__(make,model,year)
        #定义自己特有的属性和方法
        self.battery_size = 70

    def describe_battery(self):
        print("This car has a " + str(self.battery_size) + "kWh battery.")

my_tesla = ElectricCar('tesla','model s','2016')
print(my_tesla.get_descriptive_name())

my_tesla.describe_battery()
#输出结果
2016 tesla model s
This car has a 70kWh battery.

- 重写父类的方法

对于父类的方法，当他不再符合子类的需要时，可在子类中对其进行重写。重写后的父类方法需要与原父类方法同名，python在执行该方法的时候优先考虑的是子类中的方法。

在Car类中:

def fill_gas_tank(self):
        print("There are "+ str(self.gas) + " in this car")

在ElectricCar类中：

def fill_gas_tank(self):
        print("This car doesn't need a gas tank!")

使用实例时:

my_tesla = ElectricCar('tesla','model s','2016')
my_tesla.fill_gas_tank()
#输出结果
This car doesn't need a gas tank!

以上即实现了对父类方法的重写

- 将实例用作属性

即，将一个类赋值给另一个类中的某个属性

#使用类和实例
class Car():
    def __init__(self,make,model,year):
        self.make = make
        self.model = model
        self.year = year
        self.odometer_reading = 0

    def get_descriptive_name(self):
        long_name = str(self.year) + ' ' + self.make + ' ' + self.model
        return long_name

    def read_odometer(self):

    def updata_odometer(self,mileage):
        self.odometer_reading = mileage

class Battery():
    def __init__(self,battery_size=70):
        self.battery_size = battery_size

    def describe_battery(self):
        print("This car has a " + str(self.battery_size) + " kWh battery.")

class ElectricCar(Car):
    def __init__(self,make,model,year):
        #将父类与子类关联起来，让子类包含父类的所有属性，方法
        super().__init__(make,model,year)
        self.battery = Battery()

my_tesla = ElectricCar('Car','Model s','2020')
print(my_tesla.get_descriptive_name())
my_tesla.battery.describe_battery()
