函数

1. 定义函数

def display_message():
    print("I have learned the function in this section")

display_message()
#输出结果
I have learned the function in this section

​ 在python中定义函数用的是def语句

2. 向函数传递参数

def display_message(animal_type, pet_name):
    print(animal_type)
    print(pet_name)

display_message('Dog', 'hurry')
#输出结果
Dog
hurry

要注意函数的参数顺序一定要正确！

但是如下时，可以不注意函数参数顺序

关键字实参

def display_message(animal_type, pet_name):
    print(animal_type)
    print(pet_name)

display_message(pet_name = 'hurrt',animal_type = 'Dog')
#输出结果
Dog
hurrt

若函数如此

def display_message(pet_name, animal_type = 'Dog'):
    print(animal_type)
    print(pet_name)

display_message(pet_name = 'hurrt')
#输出结果
Dog
hurrt

则是默认animal_type为dog，即该函数有默认值

但需要注意的是，该函数的参数顺序需要一致

3. 返回值

python中的返回值和C中差不多

def get_message(first_name, last_name):
    full_name = first_name + last_name
    return full_name.title()

musician = get_message('wang','weijun')
print(musician)
#输出结果
Wangweijun

让实参变为可选的

即，函数可以根据输入实参的数目来进行具体的操作

其原理是python会将非空字符串解读成True

def get_message(first_name, last_name, middle_name = ''):
    if middle_name:
        full_name = first_name + ' ' + middle_name + ' ' +last_name 
    else:
        full_name = first_name + ' ' + last_name
    return full_name.title()

musician = get_message('wang','weijun','NB')
print(musician)
musician = get_message('wang','weijun')
print(musician)
#输出结果
Wang Nb Weijun
Wang Weijun

5. 传递列表

def greet_user(names):
    for name in names:
        msg = name.title()
        print(msg)

usernames = ['abc','nice','wocao']

greet_user(usernames)
#输出结果
Abc
Nice
Wocao

​ 即，将列表作为参数即可

6.修改列表中的参数

def print_models(unprinted_designs,completed_models):
    while unprinted_designs:
        current_design = unprinted_designs.pop()
        print("printing model: " + current_design)
        completed_models.append(current_design)

def show_completed_models(completed_models):
    print("\nThe following models have been printed:")
    for completed_model in completed_models:
        print(completed_model)

unprinted_designs = ['iphone case','robot','fuck!!!']
completed_models = []

print_models(unprinted_designs,completed_models)
show_completed_models(completed_models)
#输出结果
printing model: fuck!!!
printing model: robot
printing model: iphone case

The following models have been printed:
fuck!!!
robot
iphone case

在python中，对函数传递进去的列表是可以直接修改的，不像C语言使用指针，想要保存原来列表中的值，可以采用使用列表副本的方式

7. 禁止函数修改列表

def print_models(unprinted_designs,completed_models):

8. 结合使用位置实参和任意参数实参

如果要让函数接收不同类型的实参(位置实参，关键字实参，任意数量实参)，必须在函数定义中将接纳任意数量实参的形参放在最后。

python先匹配位置实参和关键字实参，再将余下的实参都收集到最后一个形参中。所以任意数量形参在函数中以元组形式存在

def make_pizza(size,*toppings):#*toppings是作为存储任意实参的作用
    print("\nMaking a" + str(size) + 
            "-inch pizza with the following toppings:")
    for topping in toppings:
        print("- " + topping)

make_pizza(16,'pepperoin')
make_pizza(12,'fuck','tomato','yes!!!')
#输出结果
Making a16-inch pizza with the following toppings:
- pepperoin

Making a12-inch pizza with the following toppings:
- fuck
- tomato
- yes!!!

10. 使用任意数量实参

有时，函数需要接受任意数量的实参，这时，可将函数编写成能够接受任意数量的键值对

def build_profile(first,last,**user_info):
    profile = &#123;&#125;
    profile['first_name'] = first
    profile['last_name'] = last
    for key,value in user_info.items():
        profile[key] = value
    return profile

user_profile = build_profile('albert','einstein',location = 'princeton',field = 'physics')
#相其中填充入新的键值对
print(user_profile)
#输出结果
&#123;'first_name': 'albert', 'last_name': 'einstein', 'location': 'princeton', 'field': 'physics'&#125;

11. 将函数存储在模块中

要让函数是可导入的，首先得创建模块。模块是扩展名为.py的文件，包含要导入程序中的代码。

pizza.py文件

def make_pizza(size, *toppings):
    print("\nMaking a " + str(size) + 
        "-inch pizza with the following toppings:")
    for topping in toppings:
        print("- " + topping)

making_pizzas.py文件

import pizza#使用import打开pizza文件，并将其中所有的函数都复制到这个程序中

#导入函数
pizza.make_pizza(16, '123123')
pizza.make_pizza(12, 'mushroom','green peppers','extra cheese')
#输出结果
Making a 16-inch pizza with the following toppings:
- 123123

Making a 12-inch pizza with the following toppings:
- mushroom
- green peppers
- extra cheese

12.导入特点的函数

类似的，还可使用from module_name import function_name

from pizza import make_pizza

make_pizza(16, '123123')
make_pizza(12, 'mushroom','green peppers','extra cheese')

其输出结果也是相同的
